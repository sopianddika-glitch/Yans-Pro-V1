# Kontribusi

Terima kasih telah tertarik untuk berkontribusi pada aplikasi Money Manager & Expense Tracker!

## Cara Berkontribusi
1. Fork repository ini dan buat branch baru untuk fitur/bugfix Anda.
2. Pastikan kode mengikuti standar penulisan dan struktur project.
3. Tambahkan deskripsi yang jelas pada setiap pull request.
4. Sertakan screenshot/gif jika ada perubahan pada UI.
5. Pastikan aplikasi tetap dapat dijalankan tanpa error.

## Standar Kode
- Gunakan penamaan variabel dan fungsi yang deskriptif.
- Pisahkan logika bisnis, UI, dan data model.
- Tambahkan komentar pada bagian kode yang kompleks.

## Laporan Bug & Saran
Silakan buat issue baru untuk melaporkan bug atau mengusulkan fitur.
